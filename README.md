# SessionKit

Monorepo for SessionKit packages.

## Packages

- `@sessionkit/core`
- `@sessionkit/redis`
- `@sessionkit/express`
- `@sessionkit/hono`

## Development

```bash
pnpm install
pnpm build
pnpm typecheck
pnpm test
```

## Release

```bash
pnpm changeset
pnpm version-packages
pnpm release
```
